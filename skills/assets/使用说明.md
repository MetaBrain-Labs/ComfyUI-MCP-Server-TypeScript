# ComfyUI 工作流代理技能 - 使用说明

## 技能概述

这个技能让 Claude 能够智能地处理 ComfyUI 图像/视频生成请求。当用户请求生成图片时,Claude 会自动:
1. 识别生成意图
2. 读取 MCP 操作指南
3. 按照正确的流程调用 ComfyUI MCP 工具
4. 完成图像/视频生成

## 安装方法

1. 下载 `comfyui-workflow-agent.skill` 文件
2. 在 Claude.ai 界面中上传此 .skill 文件
3. 技能将自动激活

## 工作原理

### 触发条件

当用户说以下内容时,技能会自动触发:
- "生成一张图片..."
- "创建一个视频..."
- "用 ComfyUI 做一张..."
- "我想要使用 ComfyUI 生成..."
- 任何涉及 AI 图像/视频生成的请求

### 执行流程

技能会引导 Claude 按照 4 个阶段执行:

**阶段 1: 发现与分析**
- 理解用户意图
- 调用 `comfy_list_workflows` 查找合适的工作流
- 如果需要,调用 `comfy_list_models` 验证模型名称

**阶段 2: 挂载工具**
- 调用 `comfy_mount_dynamic_tool` 注册工作流
- 提供唯一的扩展名称
- **重要**: 在这个阶段后停止,等待下一轮对话

**阶段 3: 执行**
- 使用新创建的 `run_workflow_<扩展名>` 工具
- 提交用户参数
- 获得任务 ID

**阶段 4: 监控与交付**
- 使用 `comfy_get_task_result` 轮询结果
- 向用户报告状态
- 提供最终生成的图片/视频 URL

## 技能内容

技能包含两个核心文件:

1. **SKILL.md** - 主要指南
   - 包含触发条件描述
   - 快速开始指南
   - 4 阶段流程概述
   - 关键规则和可用工具列表

2. **references/comfyui_mcp_guide.md** - 详细操作指南
   - 完整的操作生命周期说明
   - 错误处理策略
   - 资源管理指南
   - 参数映射说明

## 使用示例

### 示例 1: 基础图像生成

**用户输入**:
```
我想要使用 ComfyUI 生成一张有一个女性角色加上科幻背景的图片
```

**Claude 的执行流程**:
1. ✅ 识别这是图像生成请求,触发技能
2. ✅ 读取操作指南
3. ✅ 调用 `comfy_list_workflows` 找到合适的科幻/人像工作流
4. ✅ 调用 `comfy_mount_dynamic_tool` 挂载工作流
5. ⏸️ 停止并等待
6. ✅ (下一轮) 调用 `run_workflow_xxx` 执行生成
7. ✅ 轮询结果并提供图片 URL

### 示例 2: 指定模型生成

**用户输入**:
```
用 Flux 模型创建一个赛博朋克风格的头像
```

**Claude 的执行流程**:
1. ✅ 识别请求和指定的 Flux 模型
2. ✅ 读取操作指南
3. ✅ 调用 `comfy_list_workflows` 找到 Flux 相关工作流
4. ✅ 调用 `comfy_list_models` 验证 Flux 模型的确切文件名
5. ✅ 挂载工作流
6. ⏸️ 停止等待
7. ✅ (下一轮) 使用验证过的模型名称执行
8. ✅ 交付结果

## 关键特性

✅ **自动触发** - 无需手动激活,识别生成请求即触发
✅ **严格流程** - 强制执行正确的 4 阶段工作流
✅ **错误处理** - 内置常见错误的解决方案
✅ **模型验证** - 自动验证模型文件名避免错误
✅ **资源检查** - 在资源密集任务前检查系统状态

## 技术细节

- **技能类型**: 工作流编排
- **主要功能**: MCP 工具调用指导
- **文件大小**: ~17KB
- **依赖**: ComfyUI MCP 服务器和相关工具

## 维护和更新

如果 ComfyUI MCP 工具或工作流有更新:
1. 编辑 `references/comfyui_mcp_guide.md` 更新操作指南
2. 如果触发条件变化,更新 SKILL.md 的 description
3. 重新打包技能文件

## 故障排除

**问题**: 技能没有触发
- **解决**: 确保用户请求明确包含图像/视频生成意图

**问题**: 工作流执行失败
- **解决**: 技能会自动引导 Claude 读取错误处理指南

**问题**: 找不到模型
- **解决**: 技能会引导 Claude 调用 `comfy_list_models` 验证

## 总结

这个技能通过提供清晰的操作指南和严格的工作流程,让 Claude 能够可靠地处理 ComfyUI 图像生成任务。用户只需正常提出生成请求,技能会自动处理所有技术细节!
