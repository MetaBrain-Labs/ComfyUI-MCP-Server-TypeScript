# ComfyUI-MCP-Server - ComfyUI Model Context Protocol Integration

**ComfyUI-MCP-Server** 通过模型上下文协议（MCP）将 **ComfyUI** 与 **MCP Client**（例如：`Claude Desktop`、`Trae`、`Dify`等） 连接起来，使**MCP Client**能够直接与**ComfyUI**交互并控制它。
这种集成支持**MCP Client**识别用户意图调用。

**我们没有官方网站。你在网上看到的任何网站都是非官方的，与该项目无关。使用它们需自行承担风险。**
